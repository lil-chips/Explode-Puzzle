===== Slide 32 =====
What is virtualization?
https://www.youtube.com/watch?v=NmrJlWDG0yk
Virtualization is the creation of a virtual (rather than actual) version of something, such as an operating system, a server, a storage device or network resources and hides the physical characteristics of computing resources from their users.
Server virtualization enables different operating systems to share the same hardware and make it easy to move operating systems between different hardware, all while the applications are running.
Storage virtualization does the same thing for data. Storage virtualization creates the abstraction layer between the applications running on the servers, and the storage they use to store the data.
===== Slide 33 =====
What is hypervisor?
A hypervisor (virtual machine monitor) is a piece of computer software, firmware or hardware that creates and runs virtual machines.
===== Slide 34 =====
Virtual Machine
34
A virtual machine (VM) is an operating system OS and application environment that is installed on hypervisor which imitates dedicated hardware. The end user has the same experience on a virtual machine as they would have on dedicated hardware.
===== Slide 35 =====
Virtual Machine Migration
Virtual machine migration is one of the most important benefits of server virtualization technology. The ability to migrate a virtual machine (VM) from one physical host to another can significantly boost an organization's disaster recovery efforts and improve business agility.
===== Slide 36 =====
Virtual Machine Snapshot
A VM snapshot is a copy of the virtual machine's disk file (VMDK) at a given point in time. Snapshots are used to restore a VM to a particular point in time when a failure or system error occurs.
===== Slide 37 =====
Virtual machine management
Virtual Machine Manager allows users to:
create, edit, start and stop VMs
view and control of each VM's console
see performance and utilization statistics for each VM
view all running VMs and hosts, and their live performance or resource utilization statistics.

===== Slide 45 =====
Security services
Confidentiality – ensures that the information in a computer system and transmitted information are accessible only for reading by authorized parties.
Authentication – ensures that the origin of a message or electronic document is correctly identified, with an assurance that the identity is not false.
Integrity – ensures that unauthorized parties are not able to modify computer system assets and transmitted information.
Non-repudiation – ensures that neither the sender nor the receiver of a message be able to deny the transmission.

===== Slide 60 =====
Caesar Cipher
The famous Roman commander-in-chief and statesman Juliu Caesar (100-44 B.C.)
The Caesar cipher involves replacing each letter of the alphabet with letter standing three places further down the alphabet.
Plain = PLEASE CONFIRM RECEIPT
Cipher = SOHDVH FRQILUP UHFHLSW
Plaintext:    a b c d e f g h i  j  k l m n o p q  r s  t  u v w x y z
Ciphertext: d e f  g h i j k l m n o p q  r s  t  u v w x y  z a b c
Key = 3
