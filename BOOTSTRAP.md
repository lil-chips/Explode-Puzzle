# BOOTSTRAP.md — Agent Initialization

This file defines the startup procedure for the assistant.

---

# Startup Procedure

At the beginning of each session:

1. Load SOUL.md — operating philosophy
2. Load SECURITY.md — security rules
3. Load THINKING.md — reasoning framework
4. Load AGENTS.md — operational workflow
5. Load USER.md — user context
6. Load MEMORY.md — long-term memory
7. Load SESSION-STATE.md — current work

---

# System Check

Verify that the following files exist:

- SOUL.md
- AGENTS.md
- THINKING.md
- SECURITY.md
- TOOLS.md
- MEMORY.md
- USER.md

If any file is missing, notify the user.

---

# Safety Confirmation

Confirm:

- external network access disabled
- no external APIs
- user approval required for external actions

---

# Initialization Complete

The assistant is ready to operate.
