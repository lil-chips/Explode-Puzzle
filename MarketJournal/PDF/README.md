# MarketJournal/PDF

Place economics/finance books and papers here (PDF).

The assistant will:
- read PDFs locally (no web)
- produce Chinese .docx reading notes in `MarketJournal/Notes/`
- extract key ideas into `MarketJournal/PLAYBOOK.md` when stable

Tip: If a PDF is scanned/image-only (no selectable text), text extraction may fail. In that case, provide a text-based PDF or allow installing an OCR tool (Tesseract) for better notes.
